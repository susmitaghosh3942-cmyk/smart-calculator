🧮 Smart Calculator

A Full Stack Smart Calculator built using HTML, CSS, JavaScript, and Python Flask.

🚀 Features
Basic Arithmetic Operations (+, -, ×, ÷)
Square Root (√)
Square (x²)
Dark & Light Mode
Calculation History
Permanent History Storage
Keyboard Support
Responsive User Interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Python
Flask
📂 Project Structure
▶️ How to Run
Install Flask
Run the backend
Open frontend/index.html in your browser.
